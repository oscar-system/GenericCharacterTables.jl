GenericCharacterTables.printclassparam — Function
printclassparam([io::IO], t::CharTable, class::Union{Int64, Nothing}=nothing)

Print to io (or to the default output stream stdout if io is not given) the parameters of the class type class of the table t.

This includes the parameter names, ranges and exceptions. Leaving class unspecified will print the parameters of all character types.

Examples

julia> g=genchartab("GL2");

julia> printclassparam(g)
1	i ∈ {1,…, q - 1}
2	i ∈ {1,…, q - 1}
3	i ∈ {1,…, q - 1}, j ∈ {1,…, q - 1} except i - j ∈ (q - 1)ℤ
4	i ∈ {1,…, q^2 - 1} except i ∈ (q + 1)ℤ
source

Norms and scalar products

GenericCharacterTables allows you to compute norms and scalar products of class types. The results are correct for all possible combinations of parameters except possibly for those where the additionally returned exceptions apply. Those consist of multivariate polynomials with coefficients in a rational function field and are satisfied if the evaluation of this polynomial is an integer.
